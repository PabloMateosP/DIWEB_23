Iconografia

Los iconos forman una parte muy importante de nuestra página web ya que nos ayuda a comprender la información del texto más rapidamente. Así como son los principales elementos a la hora de tomar acciones en una página web como por ejemplo la flecha para volver, una x para cerrar algo, una lupa para búscar algun información ...
En este caso los principales iconos que hemos usados han sido iconos para la barra de navegación que se encuentra a la izquierda de nuestra página web. Donde se usan iconos simples para describir al público que encontrará al pulsar dicho botón. Por otro lado los principales iconos que se deben encontrar en una página web deben ser los iconos de operaciones core, que son los iconos de las operaciones generales que suelen aparecer en todas las páginas webs.

  • Como primera operación tenemos la flecha hacia atrás: ⬅
  • Tenemos la flecha para dar a siguiente ➞
  • Aparecen las flechas para dar arriba ⬆ y abajo ⬇
  • Tenemos el icono para la opción de enviar 📤
  • Así como el icono de guardar 💾 el cual es una tarjeta de memoria
  • El de eliminar aparece como una papelera 🗑
  • Aparece el icono de buscar como una lupa 🔎
  • Estos tres últimos iconos no he encontrado su referencia.
  • El icono de filtrar
  • El icono de ajustes el cual suele ser un engranaje
  • Y el icono de alerta como una señal de peligro
